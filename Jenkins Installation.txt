## **Option 1: Install Jenkins on Local (without Docker)**

### **Step 1: Install Java**

Jenkins needs Java (Java 11 or Java 17 recommended).

* **Ubuntu/Debian**:

```bash
sudo apt update
sudo apt install openjdk-17-jdk -y
java -version
```

* **CentOS/RHEL**:

```bash
sudo yum install java-17-openjdk -y
java -version
```

* **Windows**:
  Download and install [Java JDK 17](https://adoptium.net/temurin/releases/).

---

### **Step 2: Install Jenkins**

* **Ubuntu/Debian**:

```bash
curl -fsSL https://pkg.jenkins.io/debian/jenkins.io-2023.key | sudo tee \
  /usr/share/keyrings/jenkins-keyring.asc > /dev/null

echo deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc] \
  https://pkg.jenkins.io/debian binary/ | sudo tee \
  /etc/apt/sources.list.d/jenkins.list > /dev/null

sudo apt update
sudo apt install jenkins -y
```

* **CentOS/RHEL**:

```bash
sudo wget -O /etc/yum.repos.d/jenkins.repo \
  https://pkg.jenkins.io/redhat-stable/jenkins.repo
sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io-2023.key
sudo yum install jenkins -y
```

* **Windows**:
  Download Jenkins **.msi installer** → [Jenkins for Windows](https://www.jenkins.io/download/).
  Run installer and follow steps.

---

### **Step 3: Start Jenkins**

* **Linux**:

```bash
sudo systemctl enable jenkins
sudo systemctl start jenkins
sudo systemctl status jenkins
```

* **Windows**:
  Jenkins runs as a Windows service automatically after installation.

---

### **Step 4: Open Jenkins in Browser**

Open:

```
http://localhost:8080
```

---

### **Step 5: Get Initial Admin Password**

* **Linux**:

```bash
sudo cat /var/lib/jenkins/secrets/initialAdminPassword
```

* **Windows**:
  Check file:

```
C:\Program Files\Jenkins\secrets\initialAdminPassword
```

Copy the password → paste in browser → install suggested plugins → create first admin user → Jenkins UI is ready ✅

---

## **Option 2: Run Jenkins with Docker (Fastest)**

If you want **super quick setup**:

```bash
docker run -d -p 8080:8080 -p 50000:50000 --name jenkins \
  -v jenkins_home:/var/jenkins_home jenkins/jenkins:lts
```

Then open in browser:

```
http://localhost:8080
```

Get password:

```bash
docker exec -it jenkins cat /var/jenkins_home/secrets/initialAdminPassword
```
